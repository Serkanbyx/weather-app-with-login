# Weather App With Login

A full-stack weather application with user authentication built using the MERN stack.

## Tech Stack

- **Frontend:** React, Vite, TailwindCSS v4, React Router
- **Backend:** Express.js, MongoDB, Mongoose, JWT Authentication
- **API:** OpenWeatherMap

## Getting Started

### Prerequisites

- Node.js (v18+)
- MongoDB

### Installation

1. Install server dependencies:
```bash
cd server
npm install
```

2. Install client dependencies:
```bash
cd client
npm install
```

3. Configure environment variables:
```bash
cp server/.env.example server/.env
```

### Running the App

**Server:**
```bash
cd server
npm run dev
```

**Client:**
```bash
cd client
npm run dev
```
